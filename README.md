webapp menu untuk aplikasi warga dengan format html agar siap deploy.
storage penyimpanan berbasis firestore database (gratis).
untuk 30 KK warga perumahan. desain sisi clien dan admin.

untuk desain dashboard disisi client/user/warga:
dengan 4 Menu Layanan Utama yang di buat thumbnile spserti desain e-commers.
1.Pengaduan (bersifat pribadi yang hanya muncul disisi client sendiri, tidak muncul di publik dan tidak diketahui user lain)
2.Pengumuman (tempat notifikasi pengumuman yang mucul)
3.Iuran / Kas (self monitoring kas yang ada di admin dan informasi tunggakan iuran yang ada pada user tersebut. sudah bayar atau belum untuk bulan ini dan bulan bulan lalu)
4.kontak pengurus/admin. berisi tombol kontak admin (link whatsapp)
5.menu tombol panic button (dipakai saat dlam keadaan emergency/kriminal/kebakaran/dll  yang akan tampilkan dilayar semua user berbentuk dialog.)
serta tampilkan info terbaru 1 atau 2 pengumuman dan menu profil user atau client tersebut.

fitur akses
akses login dan pendaftaraan akun.
Warga di harapkan melakukan pendaftaran akun melalui akses ini.
form pendaftaran meliputi:
1.nama
2.blok_rumah
3.no_rumah
4.no_kk (opsi 2 data kk bila yang tinggal didalam 1 rumah beda no kk)
5.no_hp
6.email (untuk aktivasi akun/login)
7.password (untuk aktivasi akun/ login)

untuk desain dashboard disisi admin/pengurus:
1.pengaduan (berisi semua pengaduan dari warga perumahan agar dapat di tanggapi, aksi "sedang di tanggapi/diproses dan selesai)
2.pengumuman (berisi daftar pengumuman yang dapat di tampilkan ke aplikasi warga, termasuk mengirim notifikasi pengumuman ke warga yang belum bayar iuran)
3.iuran / kas (berisi semua riwayat inputan laporan iuran/kas warga, total saldo dan jumlah warga yang sudah bayar/belum bayar)
4.data warga (berisi data warga yang melakukan pendaftaran , datanya terkumpul semua disini dan bisa di filter untuk pencarian data warga)
5.menu tombol panic button (dipakai saat dlam keadaan emergency/kriminal/kebakaran/dll yang akan tampilkan dilayar semua user berbentuk dialog)

dan fitur akses login / logout admin.
